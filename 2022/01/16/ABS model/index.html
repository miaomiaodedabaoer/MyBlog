Agent-Based Simulation

Use known micro to investigate unknown macro

Investigate unknown micro to explain known macro

State chart

Comprises states and transitions

A transition occurs when:

​ • A trigger happens and
• Guard condition is true

Trigger types:

  • Rate: like timeout but distributed exponentially with the parameter “Rate” (rate=5 means on average 5 times per time unit)
  • Condition: when the specified condition is true
  • Message: when a message arrives
  • Timeout: after the specified timeout elapses (det / stoch)

Analysis

Risk Analysis

Most ABS models are stochastic: agents’ characteristics, behaviours, environment —- run the model multiple times

Sensitivity Analysis

• Needed when there are uncertainty about the value of a parameter

• Done by varying the uncertain parameter from min to max • Fixed steps

• Random points in between (Monte Carlo) • If more than one uncertain parameters:

• OFAT (varying one parameter while keeping other parameters constants)
